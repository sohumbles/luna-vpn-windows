# Luna 1.5.4-release

Luna 1.5.4 устанавливает обновления автоматически из самого приложения.
Скачанный установщик проверяется по SHA-256. Перед установкой отдельный
компонент создаёт резервную копию пользовательского состояния.

Сохраняются все настройки Luna, подписки, выбранный сервер, разрешение
диагностических отчётов и все исключения Split Tunneling, включая выбранные
приложения и игры по полному пути `.exe`.

Добавлен центр уведомлений с колокольчиком в боковой панели. В нём остаются
сообщения об обновлениях, подключениях и ошибках. Уведомление о новой версии
содержит кнопку установки.

Дата выпуска: 18.07.2026

# Luna 1.5.3-release

Luna 1.5.3 добавляет официальное и единственно действующее VPN-подключение
проекта Luna. Оно интегрировано в существующие функции приложения без удаления
пользовательских профилей, подписок, настроек и Split Tunneling.

Дата выпуска: 18.07.2026

## Что осталось без изменений

- импорт VLESS, VMess, Trojan и Shadowsocks;
- покупные и пользовательские подписки;
- вручную добавленные серверы;
- System Proxy и TUN;
- Split Tunneling для сайтов, IPv4/IPv6, CIDR, приложений и игр;
- выбор приложений и игр из файлов и уже запущенных процессов;
- локальные настройки, журналы, статистика и управление из трея.

# Luna 1.5.2-release

Это небольшой стабильный патч, который активирует добровольные анонимные диагностические отчёты. Отправка выполняется только после согласия пользователя и только при ошибках; чувствительные значения очищаются локально до обращения к Luna API и повторно очищаются сервером.

Дата выпуска: 15.07.2026

## Выбор уже запущенных приложений и игр

В Split Tunneling теперь не обязательно искать исполняемый файл вручную.
В карточках приложений и игр доступны отдельные кнопки выбора работающего
процесса.

Окно выбора показывает:

- имя процесса;
- PID текущего экземпляра;
- полный путь исполняемого файла;
- поиск по имени, PID и пути;
- обновление списка без закрытия окна;
- множественный выбор с помощью Ctrl и Shift.

Правило сохраняется по полному пути `.exe`, а не по временному PID. Поэтому оно
продолжает работать после завершения и повторного запуска приложения. Для игры
можно одновременно добавить launcher и основной игровой процесс.

Защищённые процессы Windows, процессы без доступного полного пути, текущий
экземпляр Luna и Xray-core не отображаются. Это не требует установки службы
или kernel-драйвера.

## Маршрутизация

- System Proxy: выбранный proxy-aware HTTP/HTTPS процесс направляется напрямую;
- TUN: тот же полный путь используется в process-aware правилах Xray;
- при невозможности определить владельца нового proxy-соединения применяется
  безопасный маршрут через VPN;
- существующие доменные, IPv4/IPv6 и CIDR-правила сохранены.

## Совместимость и приватность

- Windows 10/11 x64;
- прежние правила и импорт `luna.split.v1/v2` совместимы;
- список процессов формируется локально и не отправляется backend Luna;
- Installer и Portable не устанавливают неподписанный kernel-драйвер.

## Проверки

- PowerShell 5.1 syntax и WPF XAML;
- получение полного пути активного процесса;
- статическая проверка кнопок и обработчиков выбора процессов;
- интеграционные CONNECT-тесты direct и Xray upstream;
- IPv4/IPv6 CIDR, домены и маршрутизация по EXE;
- Installer, Portable и контрольные суммы;
- production API, страницы и загрузки сайта.
