# Отчёт о выпуске Luna 1.3.2-release

Дата: 13.07.2026

Выпуск исправляет компоновку главного экрана, отображение задержки во время автоматической проверки, выбор языка и уведомление о необходимости перезапуска после изменения языка или темы.

## Публикация

- Windows 10/11 x64 Installer;
- Windows 10/11 x64 Portable;
- репозиторий `sohumbles/luna-vpn-windows`.

## Контроль

- синтаксический разбор PowerShell;
- загрузка WPF XAML;
- unit-тесты сетевых счётчиков;
- проверка метаданных версии EXE;
- SHA-256 для Installer и Portable.
