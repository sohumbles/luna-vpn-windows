# Отчёт о выпуске Luna 1.3.4-release

Дата: 14.07.2026

Патч исправляет аварийное преобразование больших сетевых счётчиков в `System.Int32` и ограничивает автоматическое обновление задержки временем активной VPN-сессии Luna.

## Исправлено

- накопленные значения трафика больше 2 ГБ безопасно обрабатываются как `Int64`;
- автообновление задержки не запускается, пока Luna не установила VPN-соединение;
- после отключения VPN незавершённый автоматический замер не меняет отображаемое значение;
- ручная проверка сервера по-прежнему доступна без подключения.

## Публикация

- Windows 10/11 x64 Installer (рекомендуется);
- Windows 10/11 x64 Portable;
- исходники и история изменений в публичном репозитории Luna.

## Контроль выпуска

- синтаксический разбор PowerShell и загрузка WPF XAML;
- проверка вычислений с сетевыми счётчиками больше `Int32.MaxValue`;
- проверка условия автообновления для подключённого и отключённого состояния;
- проверка метаданных версии EXE;
- SHA-256 для Installer и Portable.
