# Отчёт о выпуске Luna 1.3.5-release

Дата: 15.07.2026

Патч исправляет неконтролируемое фоновое обновление карточки «Задержка выбранного сервера».

## Исправлено

- автоматическая проверка запускается только при активном VPN-сеансе Luna и включённой функции «Автообновление»;
- при выключенном автообновлении или отсутствии VPN фоновые проверки не выполняются;
- запоздавший результат фоновой проверки отбрасывается после отключения VPN или выключения функции;
- удалён второй независимый цикл проверки, который игнорировал выбор пользователя;
- ручная кнопка «Обновить» выполняет одну отдельную проверку.

## Публикация

- Windows 10/11 x64 Installer (рекомендуется);
- Windows 10/11 x64 Portable;
- официальный сайт: https://security-luna-vpn.ru/;
- Luna SpeedTest: https://security-luna-vpn.ru/speedtest.

## Контроль выпуска

- тестовая матрица из четырёх состояний VPN/автообновления;
- проверка PowerShell-синтаксиса и 18 автоматических тестов;
- проверка метаданных версии EXE;
- SHA-256 для Installer и Portable.
