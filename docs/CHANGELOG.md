# История изменений Luna

## 1.3.1-release

- исправлено перепутанное отображение значений «Без VPN» и «Через VPN» в
  карточке «Качество маршрута»;
- логика и сетевые этапы диагностики остались без изменений;
- подготовлена отдельная поставка MSIX для обновлений через Microsoft Store.

## 1.3.0-release

- добавлена диагностика качества текущего маршрута до YouTube, Discord,
  Microsoft, GitHub и Cloudflare;
- реализованы отдельные фазы DNS, TCP, VPN CONNECT, TLS-handshake, HTTP, TTFB
  и проверка ожидаемого ответа сервиса;
- добавлено сравнение доступности без VPN и через Luna;
- проверка запускается автоматически после подключения, вручную и периодически;
- ошибки DNS, тайм-аута, TLS и HTTP отображаются без подмены результата;
- возвращено название «Текущая задержка» на главном экране;
- общая статистика сети и трафик Luna берутся из реальных системных счётчиков.

## 1.2.2-release

- ручной запуск всегда открывает основное окно;
- добавлено ограничение одного экземпляра приложения;
- восстановлен публичный API сервиса Luna;
- реализован измеряющий локальный прокси и реальный трафик по приложениям.

## 1.2.1-release

- путь к `xray-config.json` теперь передаётся Xray в кавычках;
- исправлено подключение из профилей Windows с пробелом в имени;
- успешные уведомления больше не преобразуются в ложную ошибку DNS.

## 1.2.0-release

- приложение переведено на централизованный backend `security-luna-vpn.ru`;
- при запуске запрашиваются глобальная конфигурация, версия, новости, changelog
  и каталог серверов;
- `/api/servers` поддерживает `X-Luna-Client-Token`;
- серверы backend преобразуются в обычные профили Luna и сразу появляются в
  существующих списках выбора;
- последний корректный ответ сохраняется в
  `%LOCALAPPDATA%\Luna\backend-servers-cache.json`;
- сетевые ошибки и пустые ответы не удаляют рабочий список серверов;
- при недоступности сайта автоматически используется локальный кэш;
- добавлено периодическое и ручное обновление backend;
- локальный `servers.json` и подписки сохранены для обратной совместимости.

## 1.1.5-release

- исправлено отображение серверов и подписок после внедрения собственного шаблона выделения;
- строки GridView снова отображаются отдельными колонками вместо текста `@{id=...}`;
- постоянный фиолетовый маркер выбранного сервера сохранён;
- приватный тестовый VPN-профиль удалён из публичных сборок; новые установки получают серверы только из пользовательских подписок или `servers.json`;
- выполнен полный релизный аудит x64/x86, installer и portable;
- подготовлен комплект материалов и шаблоны упаковки для Microsoft Store.

## 1.1.4-release

- исключён внешний запуск PowerShell: installer и portable запускают только `Luna.exe`;
- добавлена миграция старых и повреждённых записей автозапуска;
- уведомления внутри Luna автоматически закрываются через 3 секунды;
- реализован системный трей с фирменной иконкой и переходами в подключение, статистику, настройки и сведения о приложении;
- закрытие или сворачивание окна переносит Luna в трей;
- реализованы автозапуск вместе с Windows и тихий запуск в трее;
- добавлен первый запрос согласия на будущие анонимные диагностические отчёты;
- добавлена Политика конфиденциальности и обязательное принятие при установке;
- portable-сборка умеет использовать Xray-core из каталога рядом с приложением;
- добавлены отдельные x64- и x86-поставки для Windows 10/11.

## 1.1.3-release

- полностью исправлена светлая тема: добавлены контрастные цвета текста, полей, списков, карточек, графика и боковой панели;
- выбранный сервер теперь сохраняет заметный, но аккуратный фиолетовый маркер и подсветку строки;
- текущий сервер автоматически выделяется после загрузки, поиска и обновления списка;
- незавершённые функции явно помечены как находящиеся в разработке;
- недоступные переключатели защиты заблокированы, чтобы приложение не имитировало работу ещё не реализованных функций;
- автозапуск теперь сохраняет путь к `Luna.exe`, а не к извлечённому PowerShell-сценарию;
- добавлен этот полный журнал изменений.

## 1.1.2-release

- подготовлен автономный установщик для Windows 11;
- в установщик включены Luna, Xray-core 26.3.27, базы `geoip.dat`, `geosite.dat` и `wintun.dll`;
- автоматически создаются рабочие каталоги и ярлыки;
- добавлен деинсталлятор и дополнительный выбор автозапуска;
- локальные подписки и пользовательский `state.json` не включаются в установочный пакет.

## 1.1.2

- исправлен ложный тайм-аут измерителя задержки;
- время TCP-подключения фиксируется непосредственно в момент ответа сервера;
- значения выше 999 мс отображаются как высокая задержка, а не как потеря;
- процент потерь рассчитывается только по действительно неудачным проверкам.

## 1.1.1

- полосы прокрутки оформлены под интерфейс Luna;
- кнопка сохранения настроек активируется только после изменения параметров;
- добавлены русский и английский языки;
- добавлены тёмная, светлая и автоматическая темы;
- главная страница адаптирована к изменению размера окна;
- скорость загрузки и отдачи добавлена под карточку режима;
- журнал перестаёт автоматически прокручиваться, когда пользователь читает старые записи;
- добавлена кнопка возврата к просмотру журнала в реальном времени;
- выбранный сервер сохраняется после обновления подписки;
- вспомогательный `servers.example.json` исключён из поставки.

## 1.1.0

- добавлено удаление подписок вместе с принадлежащими им профилями;
- журнал автоматически обновляется каждые 1,5 секунды;
- добавлены фильтры INFO, WARN и ERROR, поиск, экспорт и очистка журнала;
- системные окна ошибок заменены уведомлениями внутри Luna;
- добавлено действие «Исправить» для очистки DNS-кэша и переподключения;
- добавлена волновая анимация успешного подключения;
- график задержки получил шкалы времени и миллисекунд, hover-подсказки, джиттер и отображение потерь;
- добавлены разделы «Статистика», «Split Tunneling», «По приложениям», «Для экспертов» и «О программе»;
- добавлены расширенные параметры интерфейса и будущих функций защиты.

## 1.0.9

- данные сохраняются локально в `%LOCALAPPDATA%\LumaTunnel`.

## Функции в разработке

Эти возможности пока не считаются работающими и появятся в следующих обновлениях:

- полноценный TUN-режим и Split Tunneling;
- статистика трафика по отдельным приложениям;
- Sing-box, Clash Meta, Hysteria2, WireGuard и OpenVPN;
- Kill Switch;
- расширенная защита DNS, WebRTC и блокировка DNS-утечек;
- управление IPv6;
- настоящий системный трей и автоподключение;
- автоматическая проверка обновлений;
- сбор анонимной статистики;
- определение публичного IPv4 и интернет-провайдера.
