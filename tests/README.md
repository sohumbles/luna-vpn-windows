# Luna 1.3.0 — monitoring test harness

Автономные тесты для числовой и долговременной части мониторинга Luna. Harness не требует Pester, Visual Studio, сети, прав администратора или установленного Xray.

## Запуск

Из PowerShell 5.1 или PowerShell 7:

```powershell
powershell.exe -NoProfile -ExecutionPolicy Bypass -File .\Run-Tests.ps1
```

Код возврата `0` означает успешное прохождение всех тестов, `1` — хотя бы одну ошибку.

## Что проверяется

- точное принятие `2147483647`, `2147483648`, `2152589830`, `4294967296`, `1099511627776` как `Int64`;
- отклонение отрицательных, пустых и некорректных значений;
- передача BIGINT-счётчиков в JSON как десятичных строк без потери точности;
- расчёт скорости через `Int64`-дельты и `Double`;
- безопасная смена сессии и сброс системных счётчиков без отрицательного всплеска скорости;
- ограниченные очереди и FIFO-вытеснение;
- длительность, которая не обнуляет часы после суток;
- ускоренная симуляция 24 часов: 86 400 секунд, 17 280 latency-точек, 8 640 telemetry-точек, смена сессии, bounded history/offline queue и отправка пакетами до 120 элементов без дублей.

`Luna.TestCore.psm1` содержит минимальную изолированную модель ожидаемой семантики. Она предназначена для регрессионной проверки алгоритмов и не запускает графический интерфейс или VPN-процесс.
