# Luna release rules

Every Luna update must be synchronized across all public surfaces:

1. Update the application source and version.
2. Update the changelog and release notes in this repository.
3. Publish the matching Installer and Portable packages on `https://security-luna-vpn.ru`.
4. Push the source and release information to `https://github.com/sohumbles/luna-vpn-windows`.
5. Preserve all previous versions, language support, policies, documentation and downloads.
6. Never commit subscription URLs, UUIDs, API tokens, passwords, private keys or private server configuration.
7. Validate the PowerShell source, build outputs, hashes, website pages, API and download links before announcing a release.
