# Contributing

Bug reports and focused pull requests are welcome.

Before submitting:

1. Remove VPN credentials, subscription URLs and personal paths from logs.
2. Keep the existing connection button and Xray lifecycle behaviour stable.
3. Parse `src/Luna.ps1` with the PowerShell parser.
4. Test startup, connection, disconnection and single-instance handling.
5. Describe user-visible behaviour and verification steps in the pull request.
